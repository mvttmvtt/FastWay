FastWay
by: Matthew Welsh et al
(Jan 2024 - May 2024)

FastWay is an informational app developed using React, JavaScript, CSS, and HTML, designed to display fast food chains, their locations, and up-to-date menus. 
It allows users to view specific chains, find nearby locations, and browse current menus. 
Additionally, it includes admin features for adding new chains, updating menus, and deleting menu items, providing comprehensive management capabilities.
